# str-repeat

## Usage
Function to create a new string from a repeated substring.

```js
var strRepeat = require('str-repeat');

strRepeat("Hello, world!", 3);
//=> "Hello, world!Hello, world!Hello, world!"
```

## Running tests

Install dev dependencies:

```sh
$ npm i -d && npm test
```
